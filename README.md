# 言成不動產 客戶頁面確認圖

用於整理言成不動產目前已完成的前台畫面，並和報價單中的原始架構對照，做成可給客戶核對的頁面總覽與流程圖。

## 檔案

- `index.html`：客戶核對用頁面總覽
- `flow.html`：前台路徑與模組流程圖
- `quote-architecture.md`：從報價單整理出的初版頁面架構
- `images/`：正式使用的頁面截圖
- `incoming/`：可先丟新圖，再用同步腳本搬到 `images/`

- 指標商辦列表
- 指標商辦內容頁
- 租辦公列表
- 租辦公列表 / 篩選展開狀態
- 租辦公內容頁

## 使用方式

1. 把新圖丟到 `incoming/`
2. 執行 `./sync_assets.sh`
3. 視需要更新 `index.html`、`flow.html`
4. 執行 `./publish.sh`

## 快捷方式

- 雙擊直接發佈：`publish.command`
- 雙擊同步後發佈：`sync-and-publish.command`
